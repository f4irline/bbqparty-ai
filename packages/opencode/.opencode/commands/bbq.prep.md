description: Create a technical implementation plan for a Linear ticket
---

You are planning the technical implementation for Linear ticket $ARGUMENTS.

Follow these steps:

1. Move the ticket to "Planning" status using Linear MCP
2. Read the full ticket details from Linear, including any research comments
3. Create a detailed technical plan covering:
   - Files to be modified or created
   - API changes (if any)
   - Database/schema changes (if any)
   - Test strategy (unit tests, integration tests)
   - Potential breaking changes
4. Ask clarifying questions if you need decisions on:
   - Architecture choices
   - Implementation trade-offs
   - Scope clarifications
5. Document the technical plan as a comment on the ticket in Linear
6. Move the ticket to "Ready" status using Linear MCP

Be specific about what will change and how.
